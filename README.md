# Nutriglass

### Description:
Nutriglass is a calorie and food tracking web application designed to help users monitor their daily nutritional intake.

### Features:
- Track daily calorie consumption.
- Log food items and nutritional details.
- User-friendly interface for easy navigation.

### Technologies Used:
- Developed using modern web technologies (HTML, CSS, PHP, and SQL).
- Optimized for performance and user experience.
